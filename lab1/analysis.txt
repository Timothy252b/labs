Требуется разработать класс для работы с арифметической прогрессией. Данные класса представляют собой два поля: first хранит значение первого элемента прогрессии a0, second хранит знаменатель прогрессии r. Поле first должно быть вещественного типа, поле second целого положительного типа.

Класс должен содержать следующие методы. Метод инициализации Init принимает два параметра: значение первого члена и знаменатель. При передаче параметров необходимо контролировать, что знаменатель является положительным целым числом. Если передан неверный параметр, метод может установить знаменатель в значение по умолчанию или вывести сообщение об ошибке. Метод Read предназначен для ввода значений полей с клавиатуры. При вводе также необходимо проверять корректность знаменателя. Метод Show выводит на экран текущие значения полей в удобочитаемом формате.

Метод element принимает целочисленный аргумент j и возвращает значение j-го элемента прогрессии. Вычисление производится по формуле aj = first * pow(second, j). Если передан отрицательный индекс, метод должен возвращать -1 или другое значение, сигнализирующее об ошибке, поскольку для отрицательных индексов элементы прогрессии не определены.

Для демонстрации работы класса в главной функции необходимо создать объект класса Aj. Объект можно создать различными способами: статически в стеке, динамически с помощью оператора new, а также в виде массива объектов. Для тестирования методов следует вызвать Init с начальными значениями, затем показать значение элемента с заданным индексом с помощью метода element, вывести на экран состояние объекта через Show, затем вызвать Read для ввода новых значений и снова вычислить элемент прогрессии с индексом, введенным пользователем.

Дополнительно требуется реализовать внешнюю функцию make_Aj, которая получает два аргумента: первый член и знаменатель. Функция создает объект класса Aj, инициализирует его переданными значениями и возвращает созданный объект. Если переданные параметры некорректны (знаменатель не положительный), функция должна вывести сообщение об ошибке и завершить работу программы.

В программе следует продемонстрировать создание массива объектов, например, из трех элементов, с различными параметрами прогрессии, и вычислить для каждого несколько элементов.

Класс должен быть реализован в виде отдельных файлов: заголовочный файл Aj.h содержит объявление класса, файл Aj.cpp содержит определения методов, файл main.cpp содержит главную функцию и демонстрацию работы.
