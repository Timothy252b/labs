Что такое класс? Класс это пользовательский тип данных, который объединяет данные и функции для работы с этими данными. Класс служит шаблоном для создания объектов.
Что такое объект экземпляр класса? Объект это конкретный экземпляр класса, созданный в памяти и имеющий собственные значения для полей класса.
Как называются поля класса? Поля класса называются данными-членами или атрибутами класса.
Как называются функции класса? Функции класса называются методами или функциями-членами класса.
Для чего используются спецификаторы доступа? Спецификаторы доступа используются для управления видимостью и доступом к полям и методам класса из различных частей программы.
Для чего используется спецификатор public? Спецификатор public делает поля и методы доступными из любого места программы, где виден объект класса.
Для чего используется спецификатор private? Спецификатор private делает поля и методы доступными только внутри методов самого класса, скрывая детали реализации.
Если описание класса начинается со спецификатора class, то какой спецификатор доступа будет использоваться по умолчанию? По умолчанию используется спецификатор private.
Если описание класса начинается со спецификатора struct, то какой спецификатор доступа будет использоваться по умолчанию? По умолчанию используется спецификатор public.
Какой спецификатор доступа должен использоваться при описании интерфейса класса? Почему? Для интерфейса класса используется спецификатор public, потому что открытые методы определяют способы взаимодействия с объектом извне.
Каким образом можно изменить значения атрибутов экземпляра класса? Изменить значения атрибутов можно через открытые методы сеттеры, если атрибуты закрыты, или прямым обращением к полю, если оно открыто.
Каким образом можно получить значения атрибутов экземпляра класса? Получить значения атрибутов можно через открытые методы геттеры, если атрибуты закрыты, или прямым обращением к полю, если оно открыто.

Обратиться напрямую нельзя, так как поле name является закрытым по умолчанию. Необходимо использовать открытый метод для доступа.
Класс описан следующим образом class Student { string name; int group; public: ..... }; Объект класса определен следующим образом Student s; Как можно обратиться к полю name объекта s? Прямое обращение s.name невозможно, так как поле name находится в закрытой секции. Нужно использовать открытые методы геттеры, если они реализованы.
