Требуется разработать класс Pair для представления пары чисел. Первое поле должно быть целого типа, второе поле вещественного типа двойной точности. Класс предназначен для хранения двух числовых значений и выполнения операций над ними.

В классе необходимо реализовать три конструктора. Конструктор без параметров инициализирует оба поля нулевыми значениями. Конструктор с параметрами принимает целое число для первого поля и вещественное для второго и присваивает их соответствующим полям. Конструктор копирования создает новый объект путем копирования значений полей из существующего объекта. Деструктор не требует специальной реализации, так как динамическое выделение памяти не используется, однако его наличие должно быть объявлено.

Для доступа к полям класса требуются методы-селекторы, возвращающие значения полей без возможности их изменения, и методы-модификаторы, позволяющие устанавливать новые значения. Селекторы должны быть объявлены как константные, модификаторы принимают значения соответствующих типов.

Оператор присваивания должен быть перегружен для корректного копирования данных из одного объекта в другой. Реализация оператора должна копировать значения первого и второго полей и возвращать ссылку на текущий объект для поддержки цепочек присваиваний.

Необходимо перегрузить операторы потокового ввода и вывода. Оператор вывода должен выводить пару чисел в формате первое число, затем двоеточие, затем второе число, без пробелов. Оператор ввода должен считывать строку целиком, затем находить в ней позицию двоеточия. Первая часть строки до двоеточия преобразуется в целое число и сохраняется в первое поле, вторая часть после двоеточия преобразуется в вещественное число и сохраняется во второе поле. Если двоеточие не найдено, поток должен быть переведен в состояние ошибки.

В соответствии с вариантом задания требуется перегрузить операцию вычитания для пар чисел. Оператор минус должен создавать новую пару, в которой первое поле равно разности первых полей операндов, второе поле равно разности вторых полей. Также требуется перегрузить операцию добавления константы к паре с двумя вариантами. Если константа имеет целый тип, то она должна прибавляться только к первому полю. Если константа имеет вещественный тип, она должна прибавляться только ко второму полю. Для этого необходимо реализовать два разных оператора плюс: один принимает целый аргумент, другой принимает вещественный. Оба оператора не должны изменять исходный объект, а возвращать новый объект с измененными значениями.

Демонстрационная программа должна создавать объекты с помощью всех трех конструкторов: без параметров, с параметрами и копирования. Затем необходимо показать работу оператора присваивания в цепочке, например присвоить одному объекту значение другого и затем третьему. Вывести на экран содержимое объектов с помощью перегруженного оператора вывода. Продемонстрировать операцию вычитания пары из пары и операцию добавления целой константы, а также добавления вещественной константы. В конце программы показать работу оператора ввода, запросив у пользователя значения для одной или двух пар в формате число:число, и вывести введенные значения.
