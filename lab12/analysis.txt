Программа состоит из трех задач, демонстрирующих работу с ассоциативным контейнером multiset для стандартных и пользовательских типов, а также параметризованного класса на его основе.

В первой задаче используется контейнер multiset для хранения вещественных чисел. Множество с повторениями выбрано потому, что исходные данные содержат дублирующиеся значения, которые необходимо сохранить. Функция добавления заполняет контейнер элементами из вектора с помощью метода insert. Для вычисления среднего арифметического все элементы суммируются в цикле, после чего среднее значение добавляется в контейнер. Удаление элементов в заданном диапазоне выполняется прямым обходом итератора с вызовом метода erase, который возвращает следующий валидный итератор. Для добавления к каждому элементу суммы минимального и максимального значений сначала находятся эти значения обращением к begin и rbegin, затем создается новый контейнер, куда помещаются все элементы с увеличенными значениями, после чего исходный контейнер заменяется новым. Все операции сопровождаются выводом промежуточных состояний.

Во второй задаче используется тот же контейнер multiset, но для хранения объектов пользовательского типа Pair. Структура Pair содержит два целых поля. Оператор меньше перегружен так, что сравнение выполняется сначала по первому полю, затем по второму. Это необходимо для определения порядка элементов в контейнере. Оператор равенства перегружен для сравнения полей поэлементно. Оператор вывода выводит пару в круглых скобках через запятую. Исходный список содержит шесть пар, включая повторяющиеся. Для вычисления среднего арифметического суммируются отдельно первые поля и отдельно вторые, после чего создается новая пара из целых частей от деления сумм на размер контейнера. Удаление элементов в заданном диапазоне осуществляется аналогично первой задаче, но сравнение выполняется с использованием перегруженного оператора меньше. Для добавления суммы минимальной и максимальной пар находятся первая и последняя пары в упорядоченном контейнере, вычисляется новая пара, сложением соответствующих полей, затем каждый элемент увеличивается на эти значения.

В третьей задаче создается шаблонный класс ParameterizedList, который внутри использует multiset. Конструктор класса не требуется, так как элементы добавляются через метод addElements. Метод addAvgToEnd использует вызов constexpr if для различения типов. Для типа double суммируются значения элементов, среднее добавляется в контейнер. Для типа Pair суммируются отдельно первые и вторые поля, создается пара из средних арифметических, которая также добавляется. Метод deleteElementsInRange удаляет все элементы, которые находятся между low и high, используя тот же алгоритм обхода итератора. Метод addSumMinMaxToEach находит минимальный и максимальный элементы через begin и rbegin, вычисляет сумму минимального и максимального, затем для каждого типа по-разному обрабатывает добавление. Для double к каждому элементу прибавляется вычисленная сумма. Для Pair создается новая пара, где каждое поле увеличено на соответствующее поле суммы минимальной и максимальной пар. Метод print выводит все элементы контейнера.

Демонстрационная программа создает объект ParameterizedList для типа double, добавляет в него шесть значений, показывает начальное состояние. Затем добавляет среднее арифметическое, удаляет элементы в интервале от двух до пяти, и наконец прибавляет к каждому элементу сумму минимального и максимального. Аналогичные операции выполняются для ParameterizedList с типом Pair. Исходные данные содержат пять пар, выполняются добавление среднего, удаление пар в диапазоне между (2,3) и (6,7), а затем добавление суммы минимальной и максимальной пар к каждому элементу. После каждой операции выводится текущее содержимое контейнера для контроля корректности работы всех методов.
