Какой метод называется чисто виртуальным? Чисто виртуальный метод это виртуальный метод, который объявлен с присваиванием нуля в конце объявления virtual void func 0. Чем он отличается от виртуального метода? Чисто виртуальный метод не имеет реализации в классе, где он объявлен, и делает класс абстрактным. Виртуальный метод имеет реализацию и может быть переопределен в производных классах.

Какой класс называется абстрактным? Абстрактным называется класс, который содержит хотя бы один чисто виртуальный метод. Объекты такого класса создать нельзя.

Для чего предназначены абстрактные классы? Абстрактные классы предназначены для определения интерфейса и задания общей структуры, которую должны реализовать производные классы. Они служат базой для иерархии классов.

Что такое полиморфные функции? Полиморфные функции это виртуальные функции, которые могут иметь разную реализацию в разных производных классах, а вызов нужной версии определяется типом объекта во время выполнения.

Чем полиморфизм отличается от принципа подстановки? Принцип подстановки это возможность использования объекта производного класса там, где ожидается объект базового класса. Полиморфизм это способность одного и того же интерфейса иметь разные реализации, что достигается с помощью виртуальных функций и принципа подстановки.

Привести примеры иерархий с использованием абстрактных классов. Абстрактный класс Фигура с чисто виртуальным методом draw, производные классы Круг, Прямоугольник, Треугольник реализуют этот метод по-своему. Абстрактный класс Транспорт с методом move, производные Автомобиль, Самолет, Корабль.

Привести примеры полиморфных функций. Виртуальная функция draw в классе Фигура, вызываемая через указатель на базовый класс. Виртуальные функции speak в классах Животное, Собака, Кошка. Виртуальная функция calculateArea в классах Shape, Circle, Rectangle.

В каких случаях используется механизм позднего связывания? Механизм позднего связывания используется при вызове виртуальных функций через указатели или ссылки на базовый класс, когда фактический тип объекта определяется во время выполнения.
