Для чего нужен конструктор? Конструктор нужен для инициализации объекта при его создании, установки начальных значений полей класса.
Сколько типов конструкторов существует в С++? В С++ существует три основных типа конструкторов: конструктор без параметров, конструктор с параметрами, конструктор копирования. Также есть конструктор перемещения.
Для чего используется деструктор? Деструктор используется для освобождения ресурсов, которые были захвачены объектом при его работе, например для освобождения динамически выделенной памяти. В каких случаях деструктор описывается явно? Деструктор описывается явно, когда класс управляет динамическими ресурсами, такими как указатели на динамическую память, файловые дескрипторы, сетевые соединения.
Для чего используется конструктор без параметров? Конструктор без параметров используется для создания объекта с значениями по умолчанию. Конструктор с параметрами? Конструктор с параметрами используется для создания объекта с конкретными значениями, передаваемыми в качестве аргументов. Конструктор копирования? Конструктор копирования используется для создания нового объекта как копии существующего объекта.
В каких случаях вызывается конструктор копирования? Конструктор копирования вызывается при инициализации одного объекта другим, при передаче объекта в функцию по значению, при возврате объекта из функции по значению.
Перечислить свойства конструкторов. Конструктор имеет имя совпадающее с именем класса, не возвращает значение, может иметь параметры, может быть перегружен, вызывается автоматически при создании объекта, может иметь список инициализации.
Перечислить свойства деструкторов. Деструктор имеет имя совпадающее с именем класса с тильдой в начале, не возвращает значение, не принимает параметры, не может быть перегружен, вызывается автоматически при уничтожении объекта.
К каким атрибутам имеют доступ методы класса? Методы класса имеют доступ ко всем атрибутам данного класса независимо от их спецификатора доступа public, private или protected.
Что представляет собой указатель this? Указатель this это неявный указатель, который внутри метода класса ссылается на текущий объект, для которого был вызван метод. Он содержит адрес этого объекта.
Какая разница между методами определенными внутри класса и вне класса? Методы определенные внутри класса считаются встраиваемыми по умолчанию. Методы определенные вне класса с помощью оператора разрешения области видимости не являются встраиваемыми по умолчанию.
Какое значение возвращает конструктор? Конструктор не возвращает никакого значения.
Какие методы создаются по умолчанию? По умолчанию компилятор создает конструктор без параметров, конструктор копирования, деструктор и оператор присваивания.
Какое значение возвращает деструктор? Деструктор не возвращает никакого значения.
Дан класс с объявленными конструкторами и деструктором. Какой метод отсутствует в описании класса? Отсутствует оператор присваивания.
Какой метод будет вызван при выполнении операторов student s; s new student? Будет вызван конструктор без параметров.
Какой метод будет вызван при выполнении оператора student s Ivanov 20? Будет вызван конструктор с параметрами.
Какие методы будут вызваны при выполнении операторов student s1 Ivanov 20 student s2 s1? Будет вызван конструктор с параметрами для s1 и конструктор копирования для s2.
Какие методы будут вызваны при выполнении операторов student s1 Ivanov 20 student s2 s2 s1? Будет вызван конструктор с параметрами для s1, конструктор без параметров для s2, затем оператор присваивания.
Какой конструктор будет использоваться при передаче параметра в функцию print void print student a a show? Будет использоваться конструктор копирования для создания параметра a.
Класс описан следующим образом с полями name и age и открытыми методами set_name и set_age. Объект p создан. Каким образом можно присвоить новое значение атрибуту name объекта p? Через вызов метода set_name следующим образом p.set_name новое значение.
