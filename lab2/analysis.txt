Необходимо разработать класс для представления информации о заработной плате сотрудника. Класс должен называться Salary. В классе требуется определить три поля данных: фамилия и имя сотрудника в виде строки, оклад в виде вещественного числа двойной точности, а также процент премии в виде целого числа. Процент премии представляет собой долю от оклада, которая начисляется сотруднику дополнительно.

Для работы с этим классом требуется реализовать три конструктора. Конструктор без параметров должен инициализировать поля значениями по умолчанию: строку пустым значением или, например, словом empty, оклад нулем, процент премии нулем. Конструктор с параметрами должен принимать три аргумента: строку с именем, оклад и процент премии. При передаче параметров следует предусмотреть возможность передачи строки по константной ссылке для повышения эффективности. Конструктор копирования создает новый объект как копию существующего. Для этого можно использовать автоматически сгенерированный компилятором конструктор, так как все поля класса поддерживают почленное копирование.

Деструктор класса не требует специальной реализации, поскольку динамическое выделение памяти не используется. Однако его наличие необходимо объявить.

В классе необходимо определить методы доступа к полям. Это сеттеры для установки значений оклада и процента премии, а также геттеры для получения всех трех полей. Геттер для имени должен возвращать константную ссылку на строку, чтобы избежать копирования и случайного изменения данных. Геттеры для оклада и премии должны возвращать значения по значению. Все методы, которые не изменяют состояние объекта, должны быть объявлены как константные.

Также требуется реализовать метод print, который выводит на экран все данные о сотруднике в удобочитаемом формате, например, фамилию, оклад и процент премии, разделенные пробелами или другим разделителем.

Демонстрационная программа должна создавать объекты класса тремя разными способами. Первый способ - вызов конструктора без параметров, при котором создается объект с полями по умолчанию. Второй способ - вызов конструктора с параметрами для создания объекта с конкретными значениями имени, оклада и процента премии. Третий способ - использование конструктора копирования для создания нового объекта на основе уже существующего. После создания объектов программа должна показать их содержимое с помощью метода print. Затем следует изменить значение оклада у одного из объектов через метод setSalary и повторно вывести данные, чтобы продемонстрировать работу модификатора. Для каждого созданного объекта необходимо вызвать метод print, чтобы убедиться в корректной работе всех конструкторов и методов доступа.
