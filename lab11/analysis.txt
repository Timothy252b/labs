Программа состоит из пяти задач, демонстрирующих работу с различными контейнерами и адаптерами стандартной библиотеки C++.

В первой задаче используется контейнер list для хранения вещественных чисел. Исходный список инициализируется пятью значениями от полутора до пяти с половиной. Сначала вычисляется среднее арифметическое всех элементов путем суммирования и деления на размер списка. Полученное среднее добавляется в конец контейнера. Затем из списка удаляются все элементы, значения которых попадают в интервал от двух до четырех. Для этого создается вспомогательный список элементов, подлежащих удалению, и для каждого из них вызывается метод remove. После этого находятся минимальный и максимальный элементы с помощью алгоритмов min_element и max_element. Их сумма вычисляется и прибавляется к каждому элементу исходного списка. Все промежуточные состояния выводятся на экран.

Во второй задаче используется тот же контейнер list, но для хранения объектов пользовательского типа Pair. Структура Pair является шаблонной и содержит два поля одного типа. Оператор меньше перегружен так, что сравнение выполняется по сумме полей. Оператор больше перегружен аналогичным образом. Оператор равенства сравнивает оба поля поэлементно. Оператор вывода выводит пару в круглых скобках через запятую. Исходный список содержит пять пар целых чисел. Вычисляется сумма всех элементов пар, затем среднее арифметическое как вещественное число, после чего в конец списка добавляется новая пара, у которой первое поле равно среднему значению, а второе ноль. Затем удаляются все пары, у которых сумма полей находится в диапазоне от пяти до пятнадцати включительно. После этого находятся минимальная и максимальная пары по сумме полей, вычисляется сумма их значений, и эта сумма прибавляется к каждому полю каждой пары в списке.

В третьей задаче создается параметризованный класс MyList, который внутри использует контейнер list. Класс имеет конструктор от списка инициализации. Метод addAvg вычисляет среднее арифметическое элементов, суммируя их с помощью оператора сложения, и добавляет результат в конец. Метод eraseByKeyRange удаляет все элементы, значения которых попадают в заданный диапазон. Метод addMinMaxToEach находит минимальный и максимальный элементы, вычисляет их сумму и прибавляет ее к каждому элементу контейнера. Метод print выводит все элементы. В демонстрации создается объект MyList для типа double, выполняются все три операции, и результаты выводятся на экран.

В четвертой задаче используется адаптер контейнера priority_queue, настроенный на хранение объектов типа Pair с использованием критерия сравнения greater, что обеспечивает извлечение элементов в порядке возрастания суммы полей. В очередь добавляются пять пар целых чисел. Для добавления среднего арифметического все элементы извлекаются из очереди в вектор, вычисляется сумма и среднее, затем создается новая пара со средним значением и нулем, которая помещается обратно в очередь. Для удаления элементов с суммой в диапазоне от шести до десяти создается временная очередь отфильтрованных элементов. После этого из оставшихся элементов находятся минимальный и максимальный по сумме, вычисляется сумма их значений, и каждое поле каждой пары увеличивается на эту сумму. Все операции сопровождаются выводом содержимого очереди.

В пятой задаче создается параметризованный класс MyPriorityQueue, который внутри использует priority_queue с элементами Pair. Конструктор принимает список инициализации пар. Метод addAvg извлекает все элементы, вычисляет сумму полей, среднее значение и добавляет новую пару. Метод eraseByKeyRange фильтрует элементы, оставляя только те, чья сумма полей вне заданного диапазона. Метод addMinMaxToEach находит минимальный и максимальный элементы, вычисляет сумму их полей и прибавляет ее к каждому полю всех пар в очереди. В демонстрации создается очередь из пяти пар целых чисел, последовательно выполняются все три операции, и после каждой выводится текущее состояние.

Основная функция последовательно вызывает все пять задач. Каждая задача работает независимо и выводит результаты своих операций. Программа демонстрирует работу со стандартными контейнерами, адаптерами, пользовательскими типами и параметризованными классами-обертками.
