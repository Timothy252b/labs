Требуется разработать шаблонный класс-контейнер List для хранения элементов произвольного типа. Внутренняя структура списка должна быть организована в виде двусвязного списка. Для этого в классе List определяется вложенная структура Node, которая содержит поле data типа Data для хранения значения, а также указатели prev и next на предыдущий и следующий узлы.

Класс List должен хранить указатели на голову и хвост списка, а также целочисленное поле size для отслеживания количества элементов. Необходимо реализовать конструктор по умолчанию, создающий пустой список. Конструктор с целочисленным параметром n должен создавать список из n элементов, инициализируя их значением по умолчанию для типа Data. Конструктор копирования должен создавать глубокую копию переданного списка. Шаблонный конструктор копирования позволяет создавать список одного типа на основе списка другого типа с преобразованием типов элементов. Деструктор должен последовательно удалять все узлы списка.

Оператор присваивания должен выполнять проверку на самоприсваивание, очищать текущий список и копировать содержимое другого списка. Методы pushBack и pushFront добавляют элементы в конец и начало соответственно. Методы popBack и popFront удаляют элементы. Методы front и back возвращают первый и последний элементы. Метод getSize возвращает количество элементов, метод print выводит содержимое списка.

Оператор доступа по индексу должен быть перегружен для чтения и записи. Он должен проверять допустимость индекса и последовательно проходить от головы до нужной позиции. Оператор приведения к целому типу возвращает размер списка.

Внешняя шаблонная функция operator* должна выполнять поэлементное умножение двух списков разных типов. Тип результирующего списка определяется через decltype и declval как тип результата умножения элемента первого списка на элемент второго. Функция проходит по обоим спискам одновременно с помощью итераторов begin и end, умножает соответствующие элементы и добавляет их в результирующий список. Если списки разной длины, умножение выполняется по минимальной длине.

Пользовательский класс Pair является шаблонным и хранит два поля разных типов first и second. Для него должны быть реализованы конструктор по умолчанию, инициализирующий оба поля нулями, конструктор с параметрами для установки значений, шаблонный конструктор копирования, позволяющий создавать объект Pair с одними типами из объекта Pair с другими типами с преобразованием. Деструктор не требует специальной реализации.

Оператор присваивания копирует значения полей. Методы getFirst и getSecond возвращают значения полей, setFirst и setSecond изменяют их. Для работы в составе контейнера Pair должен поддерживать операции, необходимые для умножения списков. В данном случае требуется перегрузить операторы сложения с целым числом и с вещественным числом, а также оператор вычитания пары из пары. Детали этих операций определяются вариантом задания. Операторы ввода и вывода должны быть перегружены как дружественные функции. Оператор вывода выводит первое и второе поле через двоеточие.

Демонстрационная программа должна показать работу шаблона List с различными стандартными типами. Создаются списки типа List с целыми числами, List с вещественными числами. Демонстрируется поэлементное умножение списков разных типов с автоматическим определением типа результата. Программа также создает объекты Pair с разными типами параметров, демонстрирует конвертирующий конструктор копирования, операторы ввода и вывода, арифметические операции. Затем создается экземпляр шаблона List для пользовательского класса Pair, заполняется значениями, и демонстрируется работа всех операций контейнера с элементами типа Pair.
