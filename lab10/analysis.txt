Требуется разработать пользовательский класс Pair для представления пары чисел. Класс должен содержать два закрытых поля: first целого типа и second вещественного типа двойной точности. Конструктор по умолчанию инициализирует оба поля нулями. Конструктор с параметрами принимает целое число для первого поля и вещественное для второго.

Для доступа к полям необходимы методы getFirst, getSecond, возвращающие значения полей без возможности изменения, и методы setFirst, setSecond для установки новых значений.

В классе требуется перегрузить оператор вычитания пар, который создает новую пару, где каждое поле равно разности соответствующих полей операндов. Оператор сложения с целой константой должен увеличивать только первое поле и возвращать новую пару. Оператор сложения с вещественной константой увеличивает только второе поле.

Оператор меньше используется для сравнения пар. Сравнение выполняется сначала по первому полю, затем по второму. Оператор равенства проверяет совпадение обоих полей.

Операторы потокового ввода и вывода должны быть перегружены как дружественные функции. Оператор вывода выводит пару в формате первое число, затем двоеточие, затем второе число без пробелов. Оператор ввода запрашивает ввод первого и второго числа отдельно.

Для работы с файлами реализуются функции бинарного ввода вывода. Функция saveToFile открывает файл в бинарном режиме для записи, сохраняет размер вектора, затем сохраняет все объекты Pair как последовательность байтов. Функция loadFromFile читает размер, затем считывает указанное количество объектов.

Функция createAndSave запрашивает у пользователя количество пар, затем в цикле создает объекты Pair через оператор ввода и добавляет их в вектор, после чего вызывает saveToFile. Функция readAndDisplay загружает данные из файла и выводит их на экран с нумерацией.

Функция deleteLessThan принимает пороговое значение пары, введенное пользователем. Загружает вектор из файла, использует алгоритм remove_if вместе с перегруженным оператором меньше для удаления всех пар, которые меньше заданной. Сохраняет измененный вектор обратно в файл. Выводит количество удаленных записей.

Функция increaseByL запрашивает целевое значение пары и число L. Загружает вектор из файла. Для каждой пары, равной целевому значению, проверяет тип L. Если L целое число, вызывает оператор сложения с целым аргументом, который изменяет первое поле. Если L вещественное, вызывает оператор сложения с вещественным аргументом, изменяющий второе поле. Сохраняет изменения и выводит количество обработанных записей.

Функция addAfterPosition запрашивает позицию начиная с единицы и количество новых записей K. Загружает вектор. Проверяет корректность позиции. В цикле запрашивает ввод K новых пар, добавляет их во временный вектор. Использует метод insert вектора для вставки новых элементов после указанной позиции. Сохраняет измененный вектор.

В основной программе организуется меню с пунктами для вызова каждой функции. Пункт шесть завершает программу. Все операции ввода вывода сопровождаются сообщениями для пользователя. Файл для хранения данных имеет фиксированное имя pairs.dat. Программа продолжает работу до тех пор, пока пользователь не выберет выход.
